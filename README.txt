==Description==
QCer'n Pocket is a productivity extension designed for data stewards to quickly identify the Invalid Values & Update Error Notice.

Features include:

- Customize how values are displayed based on your work instructions.
- Detect and Notify minor errors.
- Highlight name format errors (approx. 80% accuracy).
- Quickly scan the entire page (approx. 75% coverage).
- Guide you to update CE if it is not updated.
- Detect inactive addresses.
- Verify other default fields and notify if any are incorrect.
- Check for duplicate affiliations (approx. 75% accuracy).

==Installation==

- Download the ZIP file containing this extension.
- Open Chrome and go to chrome://extensions/.
- Enable Developer mode (toggle in the top right).
- Click Load unpacked and select the extracted extension folder.
- Start using the extension from the toolbar!
